# CSharpApp.Api

A clean, modular RESTful API built with **.NET 9**, implementing **Minimal APIs**, **MediatR (CQRS pattern)**, and explicit `Result<T>` domain handling.

---

## 🚀 Getting Started

You can run this application locally using the **.NET 9 SDK (HTTPS)** or inside a **Docker container**.

---

### Prerequisites
* [.NET 9.0 SDK](https://dotnet.microsoft.com/download/dotnet/9.0) installed on your machine.
